生物氧化和氧化磷酸化
==========
●非光合作用生物所需的能量大多来自膜、脂肪和蛋白质等有机物的氧化。
●生物体内的氧化和自然界的燃烧，在化学本质上类似：即最终产物都是H₂O和CO₂，所释放的能量也完全相等；但二者所进行的方式却大不相同。
●糖、脂肪、protein 在细胞内的彻底氧化都经过分解代谢实现，代谢物的氧化脱氢伴随着辅酶NAD⁺和FAD还原，这些携带者H离子和电子的还原型辅酶NADH或FADH₂，将电子传递给氧时，都经历电子传递过程。
●有机分子在细胞内氧化分解成H₂O和CO₂并释放能量的过程，被称为生物氧化（biological oxidation）。
●生物氧化实际上是需氧细胞中的一系列氧化还原反应，所以又称为细胞氧化和细胞呼吸。
●生物氧化的特点：
  ▶生物氧化：必须在体温条件下进行；依赖酶的催化；有机分子发生一系列的化学变化，并在此过程中逐步氧化并释放能量。这种逐步放能的方式，不会引起体温的突然升高，并使放出的能量被有效捕获。而有机分子的体外燃烧过程需要高温，并且能量被一次性释放，产生大量的光和热。
  ▶生物氧化：在氧化过程中释放的能量一般都被贮存在一些特殊的化合物，如NAD、FAD和ATP等。电子由还原型辅酶传递到氧的过程中所释放的能量，用于合成大量的ATP，占据全部生物氧化产生能量的绝大部分。例如一个Glc分子彻底氧化时生成大约30个ATP分子，其中26个是由还原型辅酶氧化所得到的。
  
●细胞呼吸作用（完全氧化过程）可被分为三个阶段：
  ▶第一阶段：生物大分子包括糖、氨基酸、脂肪酸被降解形成乙酰-CoA。乙酰-CoA是TCA氧化的二碳片段碳源。
  ▶第二阶段：TCA，乙酰-CoA氧化产生CO₂和大量的携带H离子和电子的还原性辅酶NADH和FADH₂。
  ▶第二阶段：NADH和FADH₂进入电子传递和氧化磷酸化，形成大量的ATP。
  
●*氧化磷酸化（oxidative phosphorylation）：是指NADH和FADH₂上的电子通过一系列的电子传递载体，传递个O₂，并利用所释放的能量使ADP磷酸化形成ATP的过程。
●生物氧化作用的关键问题是：有机分子中的电子是如何转移和传递的？又是如何与分子氧结合形成水并释放能量的？
●*氧化还原反应：凡是有电子从一种物质转移到另一种物质的化学反应都被称为氧化还原反应，即电子转移反应就是氧化还原反应。
●在氧化还原反应中，提供电子的分子称为还原剂（reductatant），接受电子的分子称为氧化剂（oxidant）。一种还原剂和其失去电子后形成的氧化剂即构成一个氧化还原电子对，简称氧还电对（redox pairs）。
●氧化还原反应往往都是可逆的，物质失去电子后称为氧化型，氧化型在得到电子转变成为还原型：
  ☆Aⁿ⁺ + ne⁻ <--> A 
  ★其中，Aⁿ⁺为氧化型，即电子受体；A为还原型，为电子供体；n为电子数目，e为电子。Aⁿ⁺和A 为一对氧还电对。
●氧化剂得到电子的倾向或还原剂失掉电子的倾向称为氧化还原电势（oxidation-reduction potential）。
●标准电极势，用Eᶿ表示，指电解质溶液活度为1（质量摩尔浓度）时的电极势。任何一个氧化还原对与标准氢电极组成原电池，都可测定其标准氧化还原电势。但在实际工作中，由于氢电极使用不便，通常用简单稳定的参比电池来代替氢电极，如甘汞电极等。
●*电极势的能斯特方程（Nernst equation），可算出电极在不同溶液浓度中的电极势：
  ☆Eₙ = Eᶿ + (RT/nF)ln([电子受体]ᵃ/[电子供体]ᵇ)
  ★其中，Eₙ为待测溶液的电极势，Eᶿ为标准电极势，R为气体常数，T为298K的绝对温度，n为电极上的价变化，F为法拉第常数。

●很多情况下，氢离子也参与氧化还原反应，此时溶液中的pH值会影响整个体系的氧化还原电势。因此在标准生物化学反应条件下，pH=7测得的标准氧化还原电势用Eᶿ'表示.
●在生物体系细胞中，电子的转移可通过4种方式进行：
  ①以电子的形式之直接转移：
    ☆Fe²⁺ + Cu²⁺ ——> Fe³⁺ + Cu⁺
  ②电子以氢原子的形式转移：氢原子以一个质子和一个电子组成。
    ☆AH₂ ——> A + 2e⁻ + 2H⁺
  ③电子以氢负离子的形式从电子供体转移到电子受体；
    ☆
  ④有机还原剂直接与氧结合时的电子转移，如碳氢化合物氧化形成醇：
    ☆R-CH₃ + ½O₂ ——> R-CH₂-OH
●在生物化学中，常用的“一个还原当量（reducing equivalent）”，指参加到氧化还原反应中的是一个电子的转移，不管电子以氢原子或电子形式转移。一个氢负离子为两个还原当量。
●生物体内的氧化还原对进行氧化还原反应时，基本原理和化学电池一样，可把生物体内的氧化剂和还原剂做成化学电池。任何的氧化还原对都有其特定的标准电势Eᶿ'（测定条件：pH=7.0，25℃，电子供受体浓度为1mol/L）。
●标准还原电势具有较大负值的氧化还原点对，相对较小负值电对更倾向于失去电子；标准还原电势的正值越大，越倾向于获得电子。
●*氧化还原反应对生物体非常重要：不仅是因为生物体内的很多重要反应都属于氧化还原反应，还因为生物体所需的能量基本都是来源于体内所进行的氧化还原反应。
  ◆NADH/NAD⁺的标准电势为-0.32V，倾向于将电子传递给氧还电对FMNH₂/FMN，后者的标准电势为-0.3.
●体系自由能的变化等于体系所做的最大功的能量。将氧化还原反应看成是能做最大功的电池，那么电池所做的功在数值上等于两个电极之间的电动势和电量的乘积。当电池传递的电子为1mol时，该电池在标准状态下所做的功为：ΔGᶿ' = W = nΔEᶿ'F
  ◆其中，n为氧化还原反应中传递的电子数目，F为法拉第常数。
●生物化学反应中，氧化还原反应中的自由能变化决定于一个体系转移电子的能力。
●标准电势与化学平衡常数之间的关系，可表示为：
  ☆ΔEᶿ'=ΔGᶿ'/nF = (RT/nF)lnKeq
  ◆式中的对数关系表明，电势的微小变化都将引起平衡常数的很大变化。在温度25度条件下，若n=2，一个能够进行到99%的反应（Keq=100）只需要118mV的电势。
●在实际反应条件下，反应物和产物都不是1mol，则根据能斯特方程得出反应的实际电动势。
  ☆Eₙ = Eᶿ + (RT/nF)ln([电子受体]ᵃ/[电子供体]ᵇ)
●机体中很多反应都是靠电动势推动的，比如细胞膜表面发生的很多反应都是靠电势差推动的：脂质膜不能使游离电子自由通过，形成的跨膜离子梯度，产生电势；跨膜电势可提供能量，驱使膜上反应的进行。

●需氧细胞内，糖、脂肪、氨基酸等通过各自的分解途径所形成的还原型辅酶，包括NADH和FADH₂，通过电子传递途径被重新氧化。还原型辅基分子中的氢原子以质子形式脱下，而其电子则沿着一系列的电子载体传递，最后被传递到氧分子，形成离子型氧，与质子结合形成水。在电子传递过程中释放的自由能则使ADP磷酸化形成ATP。
●电子传递过程，指电子从还原型辅基通过一系列按照电子亲和力递增顺序排列的电子载体所构成的电子传递链，也称呼吸链，传递到氧的过程。这些电子载体都具有氧化还原作用。
●*电子传递和形成ATP过程的偶联即为氧化磷酸化作用。氧化磷酸化作用是电子沿着电子传递链传递释放自由能，并将ADP磷酸化形成ATP的全过程。

●通过电子传递途径，还原型辅基借助氧分子得以氧化并释放自由基的总过程可以表示为：
  ☆NADH + H⁺ + ½O₂ -> NAD⁺ + H₂O       ΔGᶿ' = -220.07kJ/mol 
  ☆FADH₂ + ½O₂ -> FAD + H₂O      ΔGᶿ' = -181.58kJ/mol
●NADH和FADH₂的氧化都伴随着大量自由能的释放，表面它们所带的电子对具有高的转移势能，推动电子从还原型辅酶顺坡而下，直至转移到氧分子上。这种由电子传递而释放的自由能即用于合成ATP。
●细胞对燃料物质的彻底氧化最终形成CO₂和H₂O。CO₂是通过TCA形成的，H₂O则是在电子传递过程中的最后阶段生成。
