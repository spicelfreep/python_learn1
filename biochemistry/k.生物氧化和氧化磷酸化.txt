生物氧化和氧化磷酸化
==========
●非光合作用生物所需的能量大多来自膜、脂肪和蛋白质等有机物的氧化。
●生物体内的氧化和自然界的燃烧，在化学本质上类似：即最终产物都是H₂O和CO₂，所释放的能量也完全相等；但二者所进行的方式却大不相同。
●糖、脂肪、protein 在细胞内的彻底氧化都经过分解代谢实现，代谢物的氧化脱氢伴随着辅酶NAD⁺和FAD还原，这些携带者H离子和电子的还原型辅酶NADH或FADH₂，将电子传递给氧时，都经历电子传递过程。
●有机分子在细胞内氧化分解成H₂O和CO₂并释放能量的过程，被称为生物氧化（biological oxidation）。
●生物氧化实际上是需氧细胞中的一系列氧化还原反应，所以又称为细胞氧化和细胞呼吸。
●生物氧化的特点：
  ▶生物氧化：必须在体温条件下进行；依赖酶的催化；有机分子发生一系列的化学变化，并在此过程中逐步氧化并释放能量。这种逐步放能的方式，不会引起体温的突然升高，并使放出的能量被有效捕获。而有机分子的体外燃烧过程需要高温，并且能量被一次性释放，产生大量的光和热。
  ▶生物氧化：在氧化过程中释放的能量一般都被贮存在一些特殊的化合物，如NAD、FAD和ATP等。电子由还原型辅酶传递到氧的过程中所释放的能量，用于合成大量的ATP，占据全部生物氧化产生能量的绝大部分。例如一个Glc分子彻底氧化时生成大约30个ATP分子，其中26个是由还原型辅酶氧化所得到的。
  
●细胞呼吸作用（完全氧化过程）可被分为三个阶段：
  ▶第一阶段：生物大分子包括糖、氨基酸、脂肪酸被降解形成乙酰-CoA。乙酰-CoA是TCA氧化的二碳片段碳源。
  ▶第二阶段：TCA，乙酰-CoA氧化产生CO₂和大量的携带H离子和电子的还原性辅酶NADH和FADH₂。
  ▶第二阶段：NADH和FADH₂进入电子传递和氧化磷酸化，形成大量的ATP。
  
●*氧化磷酸化（oxidative phosphorylation）：是指NADH和FADH₂上的电子通过一系列的电子传递载体，传递个O₂，并利用所释放的能量使ADP磷酸化形成ATP的过程。
●生物氧化作用的关键问题是：有机分子中的电子是如何转移和传递的？又是如何与分子氧结合形成水并释放能量的？
●*氧化还原反应：凡是有电子从一种物质转移到另一种物质的化学反应都被称为氧化还原反应，即电子转移反应就是氧化还原反应。
●在氧化还原反应中，提供电子的分子称为还原剂（reductatant），接受电子的分子称为氧化剂（oxidant）。一种还原剂和其失去电子后形成的氧化剂即构成一个氧化还原电子对，简称氧还电对（redox pairs）。
●氧化还原反应往往都是可逆的，物质失去电子后称为氧化型，氧化型在得到电子转变成为还原型：
  ☆Aⁿ⁺ + ne⁻ <--> A 
  ★其中，Aⁿ⁺为氧化型，即电子受体；A为还原型，为电子供体；n为电子数目，e为电子。Aⁿ⁺和A 为一对氧还电对。
●氧化剂得到电子的倾向或还原剂失掉电子的倾向称为氧化还原电势（oxidation-reduction potential）。
●标准电极势，用Eᶿ表示，指电解质溶液活度为1（质量摩尔浓度）时的电极势。任何一个氧化还原对与标准氢电极组成原电池，都可测定其标准氧化还原电势。但在实际工作中，由于氢电极使用不便，通常用简单稳定的参比电池来代替氢电极，如甘汞电极等。
●*电极势的能斯特方程（Nernst equation），可算出电极在不同溶液浓度中的电极势：
  ☆Eₙ = Eᶿ + (RT/nF)ln([电子受体]ᵃ/[电子供体]ᵇ)
  ★其中，Eₙ为待测溶液的电极势，Eᶿ为标准电极势，R为气体常数，T为298K的绝对温度，n为电极上的价变化，F为法拉第常数。

●很多情况下，氢离子也参与氧化还原反应，此时溶液中的pH值会影响整个体系的氧化还原电势。因此在标准生物化学反应条件下，pH=7测得的标准氧化还原电势用Eᶿ'表示.
●在生物体系细胞中，电子的转移可通过4种方式进行：
  ①以电子的形式之直接转移：
    ☆Fe²⁺ + Cu²⁺ ——> Fe³⁺ + Cu⁺
  ②电子以氢原子的形式转移：氢原子以一个质子和一个电子组成。
    ☆AH₂ ——> A + 2e⁻ + 2H⁺
  ③电子以氢负离子的形式从电子供体转移到电子受体；
    ☆
  ④有机还原剂直接与氧结合时的电子转移，如碳氢化合物氧化形成醇：
    ☆R-CH₃ + ½O₂ ——> R-CH₂-OH
●在生物化学中，常用的“一个还原当量（reducing equivalent）”，指参加到氧化还原反应中的是一个电子的转移，不管电子以氢原子或电子形式转移。一个氢负离子为两个还原当量。
●生物体内的氧化还原对进行氧化还原反应时，基本原理和化学电池一样，可把生物体内的氧化剂和还原剂做成化学电池。任何的氧化还原对都有其特定的标准电势Eᶿ'（测定条件：pH=7.0，25℃，电子供受体浓度为1mol/L）。
●标准还原电势具有较大负值的氧化还原点对，相对较小负值电对更倾向于失去电子；标准还原电势的正值越大，越倾向于获得电子。
●*氧化还原反应对生物体非常重要：不仅是因为生物体内的很多重要反应都属于氧化还原反应，还因为生物体所需的能量基本都是来源于体内所进行的氧化还原反应。
  ◆NADH/NAD⁺的标准电势为-0.32V，倾向于将电子传递给氧还电对FMNH₂/FMN，后者的标准电势为-0.3.
●体系自由能的变化等于体系所做的最大功的能量。将氧化还原反应看成是能做最大功的电池，那么电池所做的功在数值上等于两个电极之间的电动势和电量的乘积。当电池传递的电子为1mol时，该电池在标准状态下所做的功为：ΔGᶿ' = W = nΔEᶿ'F
  ◆其中，n为氧化还原反应中传递的电子数目，F为法拉第常数。
●生物化学反应中，氧化还原反应中的自由能变化决定于一个体系转移电子的能力。
●标准电势与化学平衡常数之间的关系，可表示为：
  ☆ΔEᶿ'=ΔGᶿ'/nF = (RT/nF)lnKeq
  ◆式中的对数关系表明，电势的微小变化都将引起平衡常数的很大变化。在温度25度条件下，若n=2，一个能够进行到99%的反应（Keq=100）只需要118mV的电势。
●在实际反应条件下，反应物和产物都不是1mol，则根据能斯特方程得出反应的实际电动势。
  ☆Eₙ = Eᶿ + (RT/nF)ln([电子受体]ᵃ/[电子供体]ᵇ)
●机体中很多反应都是靠电动势推动的，比如细胞膜表面发生的很多反应都是靠电势差推动的：脂质膜不能使游离电子自由通过，形成的跨膜离子梯度，产生电势；跨膜电势可提供能量，驱使膜上反应的进行。

●需氧细胞内，糖、脂肪、氨基酸等通过各自的分解途径所形成的还原型辅酶，包括NADH和FADH₂，通过电子传递途径被重新氧化。还原型辅基分子中的氢原子以质子形式脱下，而其电子则沿着一系列的电子载体传递，最后被传递到氧分子，形成离子型氧，与质子结合形成水。在电子传递过程中释放的自由能则使ADP磷酸化形成ATP。
●电子传递过程，指电子从还原型辅基通过一系列按照电子亲和力递增顺序排列的电子载体所构成的电子传递链，也称呼吸链，传递到氧的过程。这些电子载体都具有氧化还原作用。
●*电子传递和形成ATP过程的偶联即为氧化磷酸化作用。氧化磷酸化作用是电子沿着电子传递链传递释放自由能，并将ADP磷酸化形成ATP的全过程。

●通过电子传递途径，还原型辅基借助氧分子得以氧化并释放自由基的总过程可以表示为：
  ☆NADH + H⁺ + ½O₂ -> NAD⁺ + H₂O       ΔGᶿ' = -220.07kJ/mol 
  ☆FADH₂ + ½O₂ -> FAD + H₂O      ΔGᶿ' = -181.58kJ/mol
●NADH和FADH₂的氧化都伴随着大量自由能的释放，表明它们所带的电子对具有高的转移势能，推动电子从还原型辅酶顺坡而下，直至转移到氧分子上。这种由电子传递而释放的自由能即用于合成ATP。
●细胞对燃料物质的彻底氧化最终形成CO₂和H₂O。CO₂是通过TCA形成的，H₂O则是在电子传递过程中的最后阶段生成。

●以一个Glc分子的分解为例：1个Glc分子通过EMPP作用产生2个ATP、2个NADH和2个丙酮酸；1个丙酮酸通过丙酮酸脱氢酶复合体催化转变为1个乙酰-CoA，并产生1个NADH；1分子的乙酰-CoA通过TCA氧化成CO₂，并产生1个ATP或GTP、3个NADH和1个FADH₂。
●当1分子Glc氧化为CO₂时，共生成10个NADH和2个FADH₂，它们的总标准自由能为：2564.8kJ/mol。一个Glc分子完全燃烧可释放能量2564.8kJ的热，说明Glc分子氧化所释放自由能的90%都贮存在这两种还原型辅基中，通过电子传递释放，并用于合成ATP。

●电子传递链存在于原核细胞的质膜上，真核细胞的线粒体内膜上。
●在电子传递过程中，电子的传递仅发生在相邻的电子载体之间；传递方向取决于每种电子载体所具有的还原势的大小。根据氧化还原电对的标准电势ΔEᶿ'可以判断电子传递的方向，电子总是由电负性较强的氧化还原对流向具有更强正电性的氧化还原电对。
●电子从电负性流向点正性系统，伴随着自由能的降低。两者的标准还原电势之差越大，自由能的释放也就越多。
●但是，反应还必须有催化剂的催化，才能顺利进行。催化剂的作用不能改变电子传递的方向。

☛生物的化学能，本质上来讲，还是电能。电子的流动就是一种势，有势就有能量。

呼吸链概念的建立
===============
●1900-1920年，氢激活学说。Wieland 提出，氢的激活时生物氧化的关键过程，而氧分子无需激活就可以与激活的H结合，形成H₂O。
●1913年，Warburg提出，生物氧化需要一种含铁的氧化酶来激活分子氧，氧的激活时生物氧化的关键步骤。
●后来，Szent-Gyorgyi将这两种学说合并，提出在生物氧化过程中氢的激活和氧的激活都是需要的，并提出在氧化酶和脱氢酶之间起电子传递作用的是黄素蛋白类物质。
●1925年，Keilin提出，细胞色素起着连接两类酶的作用。
●随后，利用分离提纯电子链中不同的组分，并在试管中进行重组研究的方法，对阐明生物氧化机制创造有利条件。
●直到现在，有关呼吸链电子传递和ATP的生成机制还未全部阐明，有待进一步深入探讨。

●电子从NADH和FADH₂传递到O₂所经过的一系列电子载体被形象的称为电子传递链，或称呼吸链。
●呼吸链：主要由蛋白质复合体组成，大致分为4个部分，包括NADH-Q还原酶、琥珀酸-Q还原酶、细胞色素还原酶和细胞色素氧化酶。
●电子传递酶复合物中的辅基：黄素（flavins）、铁硫中心（iron-sulfur）、血红素（hemes）、和铜离子等。这些辅基都是电子载体，电子传递通过与酶分子结合的这些辅基来完成。
  ☆NADH/FADH₂ -> NADH-Q还原酶/琥珀酸-Q还原酶 -> Q -> 细胞色素还原酶 -> 细胞色素c -> 细胞色素氧化酶 -> O₂

●辅酶Q（coenzyme-Q），又称泛醌（ubiquinone），简写Q。是疏水的醌类化合物，在线粒体内膜内部可以迅速的扩散。
●辅酶Q在电子传递链中的作用是将电子从NADH-Q还原酶（复合体Ⅰ）和琥珀酸-Q还原酶（复合体Ⅱ）转移到细胞色素还原酶（复合体Ⅲ）。
  ☆NADH + H⁺ + Q(氧化型) -> NAD⁺ + QH₂(还原型)       ΔGᶿ'=-69.5kJ/mol，ΔEᶿ'=0.36V
  ☆FADH₂ + Q(氧化型) -> FAD + QH₂(还原型)            ΔGᶿ'=-2.9kJ/mol，ΔEᶿ'=0.015V

●细胞色素c（cytochrome c）：细胞色素还原酶（复合体Ⅲ）借助细胞色素c将还原型QH₂再氧化：
  ☆QH₂(还原型) + 细胞色素c(氧化型)--（细胞色素还原酶）--> 细胞色素c(还原型) + Q(氧化型)     ΔGᶿ'=-36.7kJ/mol，ΔEᶿ'=0.19V
●细胞色素氧化酶（复合体Ⅳ）催化O₂使还原型细胞色素c再氧化：
  ☆细胞色素c(还原型) + ½O₂ --（细胞色素氧化酶）--> 细胞色素c(氧化型) + H₂O    ΔGᶿ'=-112kJ/mol，ΔEᶿ'=0.58V
  
1.NADH-Q还原酶
-------------
●真核生物的NADH-Q还原酶：又称NADH脱氢酶或复合体Ⅰ，是相对分子量为880kD的蛋白质复合体，由大约46种不同的亚基组成，分别由细胞核和线粒体基因组编码。
●该酶复合体催化的第一步反应是将NADH上的两个高势能电子转移到FMN辅基上，是NADH氧化，FMN还原：
  ☆NADH + H⁺ + FMN -> NAD⁺ + FMN
●FMN既可以接受两个电子形成FMNH₂；又可接受一个电子，或由FMNH₂给出一个电子，从而形成一个稳定的半醌中间产物。
●在NADH-Q还原酶复合体上，辅基FMNH₂上的电子又转移到铁硫中心[Fe-s]上，铁硫中心[Fe-S]是NADH-Q还原酶中的第二种辅基。
●[Fe-s]中心与蛋白质结合形成铁硫蛋白，又称非血红素铁蛋白，在生物系统中的许多氧化还原反应中起着关键性的电子传递作用。
●[Fe-s]中心的类型包括：
  ⑴含1个铁原子的——FeS，Fe以四面体的形式与蛋白质的4个半胱氨酸-SH配位；
  ⑵含2个铁原子的——2Fe-2S，每个Fe分别与两个半胱氨酸-SH配位，同时与2个无机S原子相连（共2个S）；
  ⑶含4个铁原子的——4Fe-4S，每个Fe分别与一个半胱氨酸-SH配位；同时与3个无机S原子相连（共4个S）；
●在NADH-Q还原酶复合体上，NADH的电子转移到辅基FMN上，进一步经过[Fe-s]中心，转移到辅酶Q上。辅酶Q以还原型的QH₂的形式在脂膜中扩散。

2.辅酶Q
-------
●辅酶Q是一种脂溶性辅酶，存在于线粒体内膜中，可结合在膜蛋白上，也可以游离状态存在。
●在呼吸链中，辅酶Q是一种与蛋白质结合不紧密的辅酶，使得它可以在黄素蛋白和细胞色素之间能够作为一种特殊灵活的电子载体起作用。
●电子传递复合物和辅酶Q等在可流动的线粒体脂双层膜中进行局部扩散和碰撞，进行电子的传递。
●辅酶Q以不同的形式在电子传递链中起作用，在电子传递链中处于一个重要的地位：他可接受NADH-Q还原酶催化脱下的氢原子和电子，还可接受线粒体其它黄素酶类脱下的氢原子和电子，还可接受线粒体其它黄素酶脱下的电子和氢原子，包括琥珀酸-Q还原酶、脂酰-CoA脱氢酶等。

●辅酶Q中存在一个以异戊二烯为单位构成的碳氢长链，在不同的生物中其长度不同。哺乳动物中最常见的是具有10个异戊二烯单位的长链，简写为Q₁₀。在非哺乳动物中，可能只有6-8个。
●辅酶Q的异戊二烯碳氢长链称为异戊二烯链，作用是使Q成为非极性化合物，使其在线粒体内膜的脂双层中可以迅速扩散。
●辅酶Q和FMN都是NADH-Q还原酶的辅酶。辅酶Q和FMN一样，可以接受或给出一个或两个电子，能以稳定的半醌形式存在。

3.琥珀酸-Q还原酶
----------------
●琥珀酸-Q还原酶（复合体Ⅱ）是嵌在线粒体内膜中的蛋白质复合物，就是TCA中是琥珀酸氧化为延胡索酸的琥珀酸脱氢酶。
●FAD作为该酶的辅基再传递电子时不与酶分离，只是将电子传递给琥珀酸脱氢酶分子的[Fe-s]中心，再传递给辅酶Q，从而进入到电子传递链。
●NADH-Q还原酶和琥珀酸-Q还原酶中的辅酶Q已证明具有完全相同的结构和性质。
●电子从FADH₂转移到辅酶Q上的标准自由能变化很小，不足以驱使质子跨膜转运。这步反应的意义在于，将FADH₂上相对高转移势能的电子进入到电子传递链中。
  ☆FADH₂ + Q(氧化型) -> FAD + QH₂(还原型)            ΔGᶿ'=-2.9kJ/mol，ΔEᶿ'=0.015V

4.细胞色素还原酶
----------------
●细胞色素还原酶，又称为复合体Ⅲ、辅酶Q-细胞色素c还原酶、细胞色素bc1复合体等。
●细胞色素是一类含有血红素辅基的电子传递蛋白质的总称。因含有血红素，所以显现出红色或褐色。
●还原型细胞色素具有明显的可见光光谱吸收，可看到α、β、γ吸收峰。
  ⑴α吸收峰的波长随细胞色素种类的不同而不同，是区分细胞色素种类的重要指标。
  ⑵β吸收峰又称为索瑞氏（soret）光谱带。
  ⑶γ吸收峰

●1925年，Keilin最先发现细胞色素在细胞呼吸中的重要作用，并根据吸收光谱的不同将细胞色素分为a、b、c三类。
  ⑴a类α吸收峰的波长在~600nm；
  ⑵b类α吸收峰的波长在~565nm；
  ⑶c类α吸收峰的波长在~550nm；
●细胞色素几乎存在于所有生物体内。只有极少数的专性厌氧微生物缺乏这类蛋白质。
●同一种细胞色素又因为其所处的环境不同，其α吸收峰的波长位置也有一些差异。细胞色素还原酶中的细胞色素b含有两个血红素b分子：
  ⑴其中一个最大吸收峰的波长在~562nm，记为b₅₆₂或bₕ(h->H)；
  ⑵而另一个最大吸收峰的波长在~566nm，记为b₅₆₆或b˪；
●两种细胞色素b对电子的亲和力不同，主要是因为环绕它的蛋白质结构不同。

●不同类型的细胞色素中血红素分子在卟啉环上的取代基各不相同，影响铁原子的氧化还原活性。b型细胞色素的血红素是b型，也称铁-原卟啉Ⅸ。
●铁-原卟啉Ⅸ也存在于血红蛋白和肌红蛋白中。
●c型与b型的差异是血红素上的乙烯通过其双键与蛋白质的半胱氨酸-SH作用，形成硫醚键与蛋白质相连。

●细胞色素还原酶除含有细胞色素b外，还含有2Fe-2S类铁硫蛋白，以及细胞色素c₁。细胞色素b在细胞色素还原酶中以非共价形式结合，而细胞色素c₁则是以共价键与蛋白质相连。
●细胞色素还原酶个亚基在线粒体内膜中的排列：
  ◆细胞色素c₁和Fe-S蛋白位于线粒体内膜的外表面；
  ◆细胞色素b有9次跨膜区域，以稳定的α螺旋形式跨越脂双层膜。
●细胞色素还原酶中，血红素辅基的Fe原子，在电子传递中发生可逆的Fe²⁺和Fe³⁺的价态变化，在电子传递链中的作用是催化电子从QH₂转移到细胞色素c上。
●细胞色素还原酶中，QH₂上的两个高势能电子是通过两个阶段，中间形成半醌阴离子Q⁻，从而实现2个电子逐个传递给2分子细胞色素c，自己被氧化成Q。辅酶Q的这种电子传递方式称为Q循环。
  ☆QH₂ + 2cytc1(oxidized) + 2H៷⁺ --> Q + 2cytc1(reduced) + 4Hₚ⁺

5.细胞色素c
----------
●细胞色素c：可以交互的与细胞色素还原酶（复合体Ⅲ）和细胞色素氧化酶（复合体Ⅳ）接触，行使在复合体Ⅲ和复合体Ⅳ之间电子传递的作用。
●细胞色素c是一种相对分子质量为13kD的球形蛋白质，由104aa组成。
●细胞溶胶中的脱辅基细胞色素c可以跨过线粒体外膜，进入线粒体内外膜间隙，被细胞色素c合成酶将血红素和蛋白质分子-脱辅基细胞色素c结合，蛋白质分子构象发生变化，使得形成的细胞色素c不再能穿过线粒体外膜，从而被锁在线粒体的内外膜间隙中。

6.细胞色素氧化酶
----------------
●细胞色素氧化酶，又称细胞色素c氧化酶或复合体Ⅳ。哺乳动物细胞色素氧化酶的相对分子质量大约为200kD，是镶嵌在线粒体内膜上的跨膜蛋白质复合体，由13个亚基构成。
●细胞色素氧化酶共有4个氧化还原活性中心：为两个a型血红素和3个铜离子，位于亚基Ⅰ和Ⅱ上。两个血红素分别为血红素a和血红素a3.
  ⑴血红素a，和其它血红素相比，有一个甲酰基取代甲基，一个15碳氢链取代乙烯基。血红素a和蛋白质以非共价键结合。
  ⑵血红素a3
◆两个血红素a虽然在化学结构上完全相同，但处在蛋白质的不同部位，具有不同的电子亲和力。
●三个铜离子：2个为CuA，1个为CuB，也是由于处在蛋白质的不同部位，具有不同的电子亲和力。CuA的势能较低（~0.24V），CuB的势能较高（~0.34V）。
●血红素a位于亚基Ⅰ上，与位于亚基Ⅱ上的两个CuA接近；血红素a3与CuB位于亚基Ⅰ上，两者之间可能通过以S原子相连；血红素a3上的Fe原子和CuB原子构成一个双核中心，可能是O₂的结合部位。
●细胞色素氧化酶，传递电子的顺序：还原型细胞色素c将所携带的电子传递给CuA双核中心，然后再传递给血红素a，最后传递给血红素a3-CuB中的双核中心。在这里O₂经过一系列还原反应，最后生成2分子H₂O。
●氧分子是一种理想的最终电子受体，它对电子的强亲和力保证了电子传递所需的热力学驱动力。

●反应机制：还原型细胞色素c第一个电子先传递给CuB，使Cu²⁺还原冲Cu⁺；第二个电子传递给血红素a3的Fe³⁺，使得使Fe³⁺还原冲Fe²⁺；将血红素a₃-CuB双核中心还原成Fe²⁺-Cu⁺双核中心。双核中心吸引的氧分别从Fe²⁺和Cu⁺上吸引一个电子形成一个过氧中间体。
●过氧中间体再吸收一个电子和两个H⁺形成高铁中间体：
  ⑴一个O原子以-2价的形式结合到+4价的Fe原子上；
  ⑵另一个O原子以HOH的形式结合到+2价Cu上。
●高铁中间体又接受一个电子和两个H⁺，最后释放出2分子H₂O。细胞色素氧化酶的血红素a₃-CuB双核中心Fe原子核Cu原子恢复到原来的氧化态。
●这种结合方式可以避免O分子在和电子结合的过程中产生对细胞有害的超氧化负离子。
☆[Fe²⁺ Cu⁺] -> [Fe²⁺-O=O Cu⁺] -> [Fe³⁺-O⁻=O⁻ Cu²⁺] -> [Fe⁴⁺=O²⁻ Cu²⁺-HOH] -> [Fe³⁺ Cu²⁺]

电子传递链
----------
●四个复合体构成了完整的电子传递链：
☆NADH/FADH₂ -> 辅酶Q -> 细胞色素b/c1 -> 细胞色素c -> 细胞色素a/a3 -> O₂

电子传递抑制剂
-------------
●电子传递抑制剂：能阻断呼吸链中某部位电子传递链的物质。利用专一性的抑制剂选择性的阻断某个传递步骤，在测定链中各组分的氧化还原状态，是研究呼吸链顺序的一种有效的方法。
●常见的抑制剂：
  ⑴鱼藤酮（rotenone）、安密妥、杀粉蝶菌素A，阻断NADH-Q还原酶中的电子传递；
  ⑵抗霉素A（antimycin A），干扰细胞色素还原酶中电子从细胞色素b的传递作用；
  ⑶氰化物、叠氮化物（CN⁻）和CO，和血红素a3中的高铁或亚铁作用，阻断细胞色素氧化酶中的传递。

线粒体
======
●真核生物中电子传递和氧化磷酸化都是在线粒体内膜上发生的。
●线粒体普遍存在于动、植物细胞中，是需氧细胞产生ATP的主要部位。不同类型细胞中的线粒体数目和特性都不同，数目可达数百到数千。
●细胞中线粒体的位置：
  一般位于消耗ATP的结构附近，或处于细胞进行氧化作用所需燃料如脂肪滴的附近，有利于细胞对ATP的生成和利用。
●线粒体形状：随不同细胞而异，平均长度为1-2μm，宽为0.1-0.5μm。
  ⑴褐色脂肪细胞的线粒体呈球状；
  ⑵肾细胞的线粒体呈圆筒状；
  ⑶成纤维细胞的线粒体呈线状；

线粒体结构
----------
●线粒体含有两层膜及夹在两层膜中间的膜间隙。
  ⑴外膜平滑稍有弹性，大约由一半脂质和一半蛋白质构成，含有线粒体孔道蛋白，构成外膜孔道，能通过相对分子质量小于4-5kD的物质，包括质子等。
  ⑵内膜含有大于20%脂质和80%的蛋白质，蛋白质含量比细胞的其它任何膜都高。内膜是细胞溶胶和线粒体基质之间的主要屏障。
    ◆内膜形成许多向内的折叠称为嵴（cristae）。嵴的数目和结构随细胞类型不同而不同。嵴的存在大大增加了内膜的表面积，增强它产生ATP的能力。干细胞线粒体内膜的表面积相当于外膜的5倍，心脏和骨骼肌线粒体的嵴相当于干细胞的3倍，可见它们对ATP的大量需求。
    ◆内膜所包围的空间为胶状的基质（matrix）。线粒体基质的体积和组成随细胞代谢的变化而变化。
    ◆线粒体内膜的内表面分布着一层排列规则的球形颗粒，为ATP合酶，直径大于8-9nm。
    ◆内膜还含有许多涉及电子传递的蛋白质复合体，以及负责跨膜物质运输的转运蛋白，运输ADP、Pi、ATP等。
    ◆占内膜约20%的脂质构成其磷脂双层。
    ◆内膜的组成大大降低了内膜对质子的通透性，有利于跨线粒体内膜的质子梯度的形成。
●线粒体的功能概括为3个方面：
  ⒈通过TCA作用，将丙酮酸以及脂肪酸等氧化为CO₂，同时使NAD⁺和FAD还原为NADH和FADH₂。这是发生在线粒体基质或面向基质的内膜蛋白上。
  ⒉通过电子传递作用，将电子从NADH和FADH₂传递至氧分子，并同时形成跨膜质子梯度。
  ⒊通过氧化磷酸化作用，将贮存于电化学质子梯度的能量由内膜上的ATP合酶复合体合成ATP。

氧化磷酸化
==========
●氧化磷酸化和底物水平磷酸化作用有原则性区别：氧化磷酸化作用是指与电子传递链相偶联的由ADP形成ATP的过程；底物水平磷酸化是指直接将一个代谢中间产物上的磷酸基团转移到ADP分子上形成ATP的过程。
●早在1940年，Ochoa等人就测定了呼吸过程中O₂的消耗和ATP生成的定量关系。利用组织匀浆或组织切片进行研究表明，组织利用O₂的同时，ATP含量随之增加；利用放射性同位素标记的无机磷酸，测定它进入ATP中的量可测出ATP的合成量。研究结果表明，每消耗一个O₂原子合成约3个ATP，这个比例关系称为磷-氧比即P/O比。
●电子由NADH到O₂的传递过程中，有3个部位可以释放足够的能量形成一个ATP；而FADH₂的电子通过辅酶Q进入电子传递链时，P/O比值是2。

ATP合成部位
==========
●ATP合成利用的能量是将NADH和FADH₂上的电子传递给氧的过程中释放的自由能。释放自由能的部位有三处；
  ⒈复合体Ⅰ将电子从NADH传递给辅酶Q的过程；
  ⒉复合体Ⅲ将电子从辅酶Q传递给细胞色素c的过程；
  ⒊复合体Ⅳ将电子从细胞色素c传递给氧的过程。
●ATP合成是由线粒体内膜上和电子传递完全不同的分子组装体执行的，是一个多亚基的复合体，ATP合酶，或称复合体Ⅴ。ATP合酶和电子传递酶类（复合体Ⅰ-Ⅳ）完全不同。
●电子传递所释放的能量必须通过一个中间体才能被ATP合酶利用。这种能量的保存和ATP合酶对它的利用称为能量偶联或能量转换。

能量偶联假说
==========
●氧化磷酸化和电子传递之间的偶联关系已经不存在任何疑问，但是电子在传递过程中究竟是怎么促使ADP磷酸化的，还存在很多未阐明的问题。
●历史上的三种假说：化学偶联假说、结构偶联假说和化学渗透假说。
  ⒈化学偶联假说：1953年Slater最先提出，认为电子传递过程中产生一种高能共价中间物，它随后的裂解驱动ADP的磷酸化作用，如糖酵解中的ATP合成。但是目前在氧化磷酸化作用中未能找到任何一种类似的高能中间产物。
  ⒉结构偶联假说：1964年Boyer最先提出，认为电子沿电子传递链传递使线粒体内膜某些蛋白质组分发生构象变化，形成高能形式，通过ATP合成而恢复其原来的结构。尚未找到有力的实验证据。
  ⒊化学渗透假说：1961年Mitchell最先提出，认为电子传递释放的能量和ATP合成是通过跨线粒体内膜的质子梯度而偶联的。

化学渗透假说
-----------
●化学渗透假说：电子传递释放的能量驱动H⁺从线粒体基质跨内膜进入膜间隙，从而形成跨线粒体内膜的H⁺电化学梯度，这个梯度的电化学电势驱动ATP合成。
●化学渗透假说可以解释一些关键的实验结果：
  ⒈氧化磷酸化作用的进行需要封闭的线粒体内膜存在。
  ⒉线粒体内膜对H⁺、OH⁻、K⁺、Cl⁻等离子都是不通透的。
  ⒊破坏H⁺浓度梯度的形成将破坏氧化磷酸化作用。
  ⒋在分离到的线粒体内膜两侧人工制造一个质子和电化学梯度，可导致ATP合成。
●Mitchell因提出化学渗透假说而获得1978年诺贝尔化学奖。但迄今为止，能量偶联的具体机制尚未完全阐明。

质子梯度的形成
-----------
●电子传递复合体Ⅰ、Ⅱ和Ⅳ推动H⁺跨过线粒体内膜到线粒体膜间隙，线粒体膜间隙和细胞溶胶是相通的。H⁺跨膜流动造成线粒体基质的H⁺浓度低，形成负电势；而膜间隙H⁺浓度高，形成正电势，从而产生电化学梯度即电动势。
●质子泵出是个耗能过程。一个质子逆电化学梯度跨过线粒体内膜的自由能变化可用公式进行表示，其中F是法拉第常数，ΔΨ为跨膜电势差。
●质子泵出与电子传递产生自由能的偶联机制存在2种假设：氧化还原回路机制和质子泵机制。
