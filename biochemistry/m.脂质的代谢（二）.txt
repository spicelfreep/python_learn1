脂质的生物合成
~~~~~~~~~~~~~

脂质的功能
==========
脂质的功能是多种多样的；
●脂肪是「能量贮存」的主要形式；
●类脂，特别是「磷脂」和「胆固醇」是细胞膜的主要组成成分，起维持细胞的完整、区隔细胞内部不同结构的作用；
●还有一些特殊的「活性脂质」，包括激素（如性激素）、辅助因子（如维生素K等）、乳化剂（如胆汁盐）、转运体（如多萜醇）、发色团（如视黄醛）以及细胞内外的信使（如前列腺素、血栓烷等），发挥特殊的生物学功能。
有机体合成各种不同脂质的能力是非常重要的。

贮存脂质
--------
●脂肪是「动物能量贮存」的主要形式：脂质是所有营养物质中「单位质量」具有「最高能量」的化合物（38KJ/mol）。用它们来贮存能量是最有利的。1g糖只产生17kJ的能量。
●来自膳食的脂肪必须先「转化」为「贮存脂肪」。脂肪的贮存和运送是相互联系的；当需要脂肪降解提供能量时，脂肪组织中的贮存脂肪释放游离脂肪酸，并转移到肝以便分解，这个过程称为“动员（mobilization）”。
●「脂肪动员」过程需要酶的参与。【脂酶】和【磷脂酶】担负着水解脂肪的作用，释放的游离脂肪酸在线粒体中进行「分解代谢」，甘油则在细胞溶胶中降解。脂肪酸一旦从脂肪细胞中游离释出，它们就渗透穿过膜，与血清「清蛋白」结合，运送到各种组织。

过度的脂肪酸mobilization可导致发展成脂肪肝（fatty liver）；
●脂肪肝可能因「糖尿病」产生，由于胰岛素欠缺不能正常动员Glc，必须使用其它营养物质代替供给能量，引起脂质的分解代谢加剧，包括过度的脂肪酸动员和肝中过度的脂肪酸降解，导致脂肪肝。
●脂肪肝的发生可能是受「化学药品」的影响，如「四氯化碳」或「吡啶」。这些化合物破坏肝细胞，导致脂肪组织去取代它们，肝的功能逐渐丧失。
●膳食中缺乏「抗脂肪肝剂」，即「胆碱」和「甲硫氨酸」不足时，会导致磷脂酰胆碱和脂蛋白合成的缺少。脂蛋白运送的脂质来自肝。脂蛋白合成的减弱导致肝中脂质的累积，产生脂肪肝。

●当机体需要自膳食中获得能量以贮存时，「脂肪酸合成」就会发生。
●脂肪酸「降解」是氧化移去「2个碳原子」单位（乙酰-CoA）的过程；脂肪酸「合成」则通过不同的途径，使用不同的酶，发生的场所也是在细胞的不同部位。
●「脂肪酸合成」在「细胞溶胶」中进行；有一个重要的3碳单元中间体即「丙二酸单酰-CoA（丙二酰-CoA）」参与脂肪酸合成，它与脂肪酸降解完全无关。随后在需要时，脂肪酸和甘油分子进一步结合形成贮存形式的脂肪。

●「脂肪酸合成」需要「乙酰-CoA」。在线粒体中产生的乙酰-CoA必需先从线粒体中转移到「细胞溶胶」中，才能参与脂肪酸合成。但线粒体内膜对乙酰-CoA是不容许通过的，需要由一个穿梭机制进行运送，称为「三羧酸转运」体系。
●「三羧酸转运」体系：使用柠檬酸作为乙酰基的载体。在线粒体中，乙酰-CoA和草酰乙酸合成「柠檬酸」；柠檬酸跨过线粒体内膜；在细胞溶胶中，受【柠檬酸裂解酶】作用而断裂，生成「乙酰-CoA或草酰乙酸」，再被运送回线粒体。
●脂肪酸合成「起始」于乙酰-CoA转化成「丙二酰单酰-CoA」。反应在【乙酰-CoA羧化酶】催化下实现的，是「脂肪酸合成」的重要步骤。
●原核生物E.coli的【乙酰-CoA羧化酶】是3种蛋白质的复合体：
  其一是「生物素羧基载体蛋白（23kD）」，是生物素载体，生物素的羧基和蛋白的lys残基-NH₂共价连接，形成「生物胞素」。
  另外两种蛋白是【生物素羧化酶（98kD）】和【转羧酶（130kD）】。
    【羧化酶】催化生物素的羧基化；
    【转羧酶】催化生物素上的活性羧基转移给乙酰-CoA，生成「丙二酰单酰-CoA」。
●真核生物哺乳类和鸟类的【乙酰-CoA羧化酶】是「同二聚体」，其【生物素羧化酶】以及生物素羧基载体都在单一的、相对分子质量为230kD的肽链上。

脂肪酸合酶
----------
●丙二酰单酰-CoA准备好之后，脂肪酸合成的下一步反应是【脂肪酸合酶复合物】的酶促反应。
  在〔动物细胞〕中，【脂肪酸合酶复合物】含有7种酶活性和「一个酰基载体蛋白（acylcarrier protein，ACP）」，并且都定位于「1个」单一的多功能肽链；
  在〔酵母〕中，【脂肪酸合酶复合物】由ACP和6个酶组成，但定位在「2个」多功能的多肽链上，其中一个含ACP和2个酶活性，另一个含余下的4种酶活性。
  在〔Ecoll和植物〕中，【脂肪酸合酶复合物】由不同的「7个」多肽链的聚合体，其中一个是ACP，其余六链是酶。
●「酰基载体蛋白ACP」是一个相对分子质量很低的蛋白质，它的辅基是「磷酸泛酰巯基乙胺」，由「磷酸」基团与ACP的「丝氨酸残基」以〔磷脂键〕连接，另一端的「-SH」与「脂酰基」相连形成〔硫脂键〕，从而可以把「脂酰基」从一个酶反应转移到另一个酶反应，由此得到酰基载体蛋白的名称。这个长链的分子犹如「臂摆」，把底物在酶复合体上从一处的催化中心「转移」到另一处。（类似丙酮酸脱氢酶复合体的「硫辛酰赖氨酸臂」）。

●7个活性酶的催化部位分别为：
  1.乙酰-CoA：ACP「转酰」酶；
  2.丙二酰-CoA：ACP「转酰」酶；
  3.β-酮酰：ACP「合」酶；
  4.β-酮酰：ACP「还原」酶；
  5.β-羟酰：ACP「脱水」酶；
  6.烯酰：ACP「还原」酶；
  7.软脂酰：ACP「硫脂」酶；
●【脂肪酸合酶】从「乙酰-CoA」开始，每循环一次，碳链延伸「2个」碳原子单元，都是来自「丙二酸单酰-ACP」的乙酰基，并释放一个CO₂。

●动物体内【脂肪酸合酶】多出【软脂酰-ACP硫脂酶】，它催化最后生成的软脂酰-CoA的水解，转化为「软脂酰」和ACP，它是在「16个碳」的脂肪酸链合成以后方显示功能。其它的有机体没有【软脂酰-ACP硫脂酶】，而是直接利用「软脂酰-ACP」。

●【脂肪酸合酶】催化的7步反应包括：
1.启动：1.乙酰-CoA：ACP「转酰」酶；

5.脱水：5.β-羟酰-ACP「脱水」酶；
6.还原：6.烯酰-ACP「还原」酶；
7.释放：7.软脂酰-ACP「硫脂」酶；

  ☆软脂酰-ACP + H₂O ——> 软脂酰 + HS-ACP 

●软脂酸若要形成更长的脂肪酸，或引入双键，必须接受「其它酶系」的作用。
●合成一分子「软脂酸」消耗的乙酰-CoA和丙二酰单酰-CoA的总结果：
  ☆乙酰-CoA + 7丙二酰单酰-CoA + 14NADPH + 20H⁺ ——> 软脂酸 + 7CO₂ + 14NADP⁺ + 8CoA-SH + 6H₂O 
  
●从起始物「乙酰-CoA」到产物软脂酸的化学计量总反应式为：
  ☆8乙酰-CoA + 7ATP + 14NADPH + 6H⁺ ——> 软脂酸 + 14NADP⁺ + 8CoA-SH + 6H₂O + 7ADP + 7Pi

●脂肪酸合成和β氧化分解的比较：
1.发生「场所」不同：合成发生于「细胞溶胶」:氧化发生于「线粒体」；
2.都有一中间体与「载体」连接：合成载体为ACP；氧化载体为「CoA」；
3.其中有4步反应，从化学上看是逆途径，但所用的酶和辅因子都不相同：
  合成中的缩合、还原、脱水、还原；
  氧化中的氧化、水合、氧化、裂解。
4.都具有「转运」机制将「线粒体和细胞溶胶」沟通起来：
  合成中的「三羧酸转运」机制运送乙酰-CoA；
  氧化中的「肉碱载体系统」，运送脂酰-CoA；
5.都以脂肪链的「逐次、轮番」变化为特色：
  合成脂肪链每次获取「2碳」单元延伸，必须与「丙二酰单酰-CoA」缩合；
  氧化中则是以乙酰-CoA形式的「2碳」单元离去。
6.合成时从分子甲基端开始到「羧基端止」；而降解相反，从「羧基端开始」离去。
7.「羟脂基」中间体在合成中的「D-构型」，但在脂肪酸降解中则是「L-构型」。
8.合成时需要NADPH参与的「还原」途径；降解则是FAD和NAD⁺参与的「氧化」途径。

碳链的延长
----------
脂肪酸碳链延长可发生在「线粒体」或「内质网」中，细胞不同部位的碳链延长机制有所差异：
●〔线粒体〕中的延长是独立于脂肪酸合成之外的过程，是「乙酰」单元的加成和还原，即「脂肪酸降解过程的逆反应」。但在脂肪酸延长最后一步使用了还原剂NADPH，而在脂肪酸延长最后一步使用了还原剂NADPH，而在脂肪酸降解中用FAD为氧化剂。
●〔内质网〕中的延长「更活跃」。基本过程遵循「脂肪酸合成途径」，只是参与的酶有改变，CoA代替了ACP。已合成的软脂酰-CoA以「丙二酸-单酰-CoA」为2碳单元供体，由NADPH还原，经过还原、脱水、还原的步骤，形成18C产物「硬脂酰-CoA」。

碳链的去饱和
-----------
动物体内最常见的两个饱和脂肪酸——软脂酸和硬脂酸，是棕榈油酸（16，Δ⁹）和油酸（18，Δ⁹）的前体。这两个「不饱和」脂肪酸都是在Δ⁹位引入一个双键，由「脂肪酰-CoA去饱和酶」催化。在哺乳动物体内，引入双键的氧化反应需要一个由两种酶和一个细胞色素组成的「电子传递体系」参与。
●【去饱和酶】的底物脂肪酸和NADP⁺，相继经历「两个电子」的氧化作用，电子传递包括黄素蛋白（细胞色素b₅还原酶）和细胞色素b₅，二者都和【去饱和酶】存在于「光面内质网」中。分子氧作为「电子受体」参与反应。

●哺乳动物缺少能够在C9位以外引进双键的酶，因此亚油酸（18，Δ⁹˙¹²）和亚麻酸（18，Δ³˙¹²˙¹⁵）不能经生物合成得到，只能通过膳食获取，是「必需脂肪酸」。

三酰甘油的合成
-------------
三酰甘油是由两个前体合成的，「脂酰-CoA」和「3-磷酸甘油」。脂酰-CoA来自脂肪酸的活化；3-磷酸甘油有两条途径，即「糖酵解」途径形成的「磷酸二羟丙酮の还原」或「甘油」的「磷酸化」形成。
●单酰甘油或二酰甘油：由3-磷酸甘油和脂酰-CoA在【酰基转移酶】作用下相继「酯化」形成。

●三酰甘油和合成和降解，随当时体内「代谢资源需求」的改变而被调节，「激素」对其调节发挥重要作用。
●「胰岛素」促进饮食中糖类和蛋白质转化为「三酰甘油」。对于患有严重胰岛素分泌或作用缺陷的糖尿病人来说，因不能将碳水化合物与氨基酸转化为「脂肪酸」而表现为体重下降；同时，病人体内因乙酰-CoA的积累导致「酮体」形成加速。

●「脂肪组织」中三酰甘油分解产生的大约75%的脂肪酸经「血液」循环到「肝」，在肝内酯化形成三酰甘油，经血液循环并在「胞外蛋白酶」作用下分解为脂肪酸回到脂肪组织，再酯化形成三酰甘油，从而构成「三酰甘油循环」。
●胰高血糖素和肾上腺皮质激素「促进」脂肪组织中三酰甘油的分解，同时抑制糖酵解、增加糖异生，释放的脂肪酸运至骨骼肌中被氧化提供能量，被肝吸收的脂肪酸不被氧化而酯化为三酰甘油，运回脂肪组织。

●既使在饥饿状态，三酰甘油在脂肪组织中不断被合成，此时3-磷酸甘油来源于「甘油异生」途径。
●「甘油异生」途径可以看作是「缩短」的糖异生途径，从丙酮酸到草酰乙酸到磷酸烯醇式丙酮酸，再到磷酸二羟丙酮，再由胞浆内的【3-磷酸甘油脱氢酶】催化生成3-磷酸甘油。
●「甘油异生」途径的功效：
  在脂肪组织中，与游离脂肪酸再酯化偶联，控制脂肪酸释放到血液中的速率；
  在褐色脂肪组织中，控制游离脂肪酸转运到线粒体参与生热作用的速率；
  对于空腹的人，仅肝中的甘油异生就能支持足够的3-磷酸甘油的合成，使比例高达65%的脂肪酸再酯化形成三酰甘油。

甘油磷脂的合成
-------------
●「甘油磷脂」是生物膜的重要组成部分。
  首先，脂酰-CoA和3-磷酸甘油在【酰基转移酶】催化下反应形成「磷脂酸」（1,2-二酰甘油-3-磷酸）；
  磷脂酸一旦形成，很快就转化为「二酰甘油」或「CDP-二酰甘油」，后者将分头进行代谢。
●在〔E.coli〕中，磷脂酸在特定的【胞苷转移酶】的催化下，活化形成CDP-二酰甘油，以CDP作为脂质成分的载体，再进行「极性头基」的反应。

●在〔真核生物〕中，〔磷脂酰胆碱〕的生物合成起始于「胆碱」被运送入细胞。胆碱是人类膳食中不可缺少的部分，不能在动物体内生成。
  胆碱进入细胞，在【胆碱激酶】作用下磷酸化为「磷酸胆碱」；
  磷酸胆碱与CTP在【CTP：磷酸胆碱胞苷转移酶】作用下转化为「CDP-胆碱」。此酶是“限速酶”，在细胞溶胶中无活性，进入内质网后，与膜的磷脂作用后而活化。
●CDP-胆碱再与二酰甘油反应，受内质网中存在的酶催化，形成「磷脂酰胆碱」。
●〔磷脂酰乙醇胺〕的合成是以「乙醇胺」为起始物，与磷脂酰胆碱合成途径相似。

●「CDP-二酰甘油」在原核生物体内的磷脂合成中是关键的中间体，在真核生物的线粒体中也可合成磷脂酰甘油和「二磷脂酰甘油」。另外CDP-二酰甘油可导致形成〔磷脂酰肌醇〕，磷脂酰肌醇占膜中脂质的5%，还可以［4-磷酸磷脂酰肌醇］和［4,5-二磷酸磷脂酰肌醇］的形式存在。
●〔磷脂酰肌醇〕的磷酸衍生物可以在【磷脂酶】的作用下形成具有重要调控功能的产物。其中1,4,5-三磷酸肌醇参与钙在细胞内贮存的移位，并引起细胞溶胶钙对各种酶的激活；
●〔二酰甘油〕是蛋白激酶C的激活剂，催化许多蛋白的磷酸化改变他们的活性。

鞘磷脂和鞘糖脂的合成
-------------------
〔鞘磷脂〕是所有脊椎动物细胞的组成成分，在髓鞘中含量丰富，具有保护和使神经纤维绝缘的作用。在红细胞以及血浆脂蛋白中都有存在。
●〔鞘磷脂〕直接由〔神经酰胺〕生成。〔鞘磷脂〕中的主要结构是〔鞘胺醇〕，它的生物合成发生在内质网中。
  最初的起始物软［脂酰-CoA］与「Ser」在【3-酮鞘氨醇合酶】的催化下缩合形成〔3-酮鞘氨醇〕；
  然后再【3-酮鞘氨醇还原酶】的催化下，还原形成「二氢鞘氨醇」；
  「二氢鞘氨醇」的氨基和一分子「脂酰-CoA」反应，形成「N-脂酰-二氢鞘氨醇」；
  再经FAD脱氢，形成「神经酰胺」；
●「鞘磷脂」是「磷脂酰胆碱」转移「磷酸胆碱」给「神经酰胺」形成的，催化的【磷酸胆碱转移酶】位于高尔基体膜的腔侧。

〔鞘糖脂〕的合成也起始于「神经酰胺」。鞘糖脂和鞘磷脂都含有鞘氨醇，不同之处是鞘磷脂含有「磷酸胆碱」，而鞘糖脂含有「糖基」（单糖、低聚糖或其衍生物）。
●〔鞘糖脂〕的合成和O-连接的糖蛋白相似，以「糖基二磷酸尿苷」如葡糖二磷酸尿苷——UDP-Glc，为起始物，与神经酰胺反应形成「葡糖-神经酰胺」〔葡糖脑苷脂〕。同样半乳糖-神经酰胺〔半乳糖脑苷脂〕也是这样合成的。
●糖分子以「糖核苷酸」的形式转移到受体的脂质分子上，参与此反应的酶是【糖基转移酶】，对各个反应都是特异的，绝大多数都是位于高尔基体的空腔一侧。

类十二烷酸的合成
---------------
〔类十二烷酸〕是一族多样化的激素，绝大多数是由C20「多不饱和」脂肪酸——花生四烯酸衍生而来。
●花生四烯酸作为「磷脂」复合物的一部分贮存于膜中，根据需要，经专一的【磷脂酶A2】催化从磷脂复合物中释放。
●【环加氧酶】途径：
  在与「内质网」结合的双功能酶——【环加氧酶】也称【前列腺素内过氧化物合酶】的催化下实现。
  该酶具有【环加氧酶】和【过氧化物酶】双重活性，使得花生四烯酸依次转变为PGG₂和PGH₂（前列腺素G₂和H₂），再进一步反应生成其它「前列腺素」和「血栓烷」。
●【环加氧酶】可以催化「自我毁灭」，在对~400个底物进行催化后，【环加氧酶】就「不可逆失活」，但其反应机制尚且不明。

●「阿司匹林」（aspirin）对【环加氧酶】活性部位的「丝氨酸」残基进行「乙酰化修饰」，使得酶「不可逆」的失活，进而导致自PGH₂衍生出来的化合物合成受阻，包括前列腺素、前列腺环素、和血栓烷。
●「阿司匹林」是著名的「抗炎」药物，其他非固醇抗炎药物，也可抑制【环加氧酶】，但它们是「非共价」结合，不是不可逆抑制。

●【线性脂加氧酶】途径：
  【脂加氧酶】对各种类十二烷酸进行催化，在5,12或15位上插入「过氧基」。在5位插入「过氧基」，形成5-氧过氧化十二碳四稀酸（5-HPETE），
  进行脱水可以形成一个环氧化物「白三烯A4」；
  谷胱甘肽的巯基和白三烯A4反应生成「白三烯C4」；
  再水解生成「白三烯D4」；
  再水解生成「白三烯E4」；
  ●以此衍生出十多种代谢物。
●「白三烯」是某些变态反应、炎症及心血管疾病的化学介质，使毛细血管和微静脉通透性增加；它的阻断剂对免疫、发炎和过敏的治疗都有重要意义。
●【脂加氧酶】存在于白细胞、心、肺、脾、脑等组织。

胆固醇的合成
-----------
〔胆固醇〕是自然界最丰富的「甾醇」化合物，生物膜和血浆脂蛋白的重要组成部分，又是类固醇激素、维生素D3和胆汁酸的前体分子。
●〔胆固醇〕来源于「异戊二烯」。
Ⅰ.第一步：
  首先是2分子的乙酰-CoA在【硫解酶】的反向催化下形成「乙酰乙酰-CoA」；
  再进一步在HMG-CoA【合酶】催化下加入第3个乙酰-CoA，反应生成「3-羟基-3-甲基戊二酰-CoA」（HMG-CoA）；
  再在HMG-CoA【还原酶】催化下，消耗两分子NADPH，形成「甲羟戊酸」；
  ●HMG-CoA【还原酶】催化的酶促反应是对胆固醇生物合成速率「起决定作用」的一步。
Ⅱ.第二步：
  包括「甲羟戊酸」的2步〔磷酸化〕形成「5-焦磷酸甲羟戊酸」；
  再由【5-焦磷酸甲羟戊酸脱氢酶】催化〔脱羧〕生成「异戊酰焦磷酸」，
  再进一步异构化形成「二甲烯丙基焦磷酸」。
●这两类「异戊二稀焦磷酸」作为第Ⅲ步的「起始物」，合成胆固醇。
Ⅲ.第三步：
  6个「异戊二稀焦磷酸衍生物」进行一系列的缩合反应，形成开链的不饱和烃——「角鲨烯」。
Ⅳ.第四步：
  「角鲨烯」两步「环和」形成「羊毛固醇」，形成类固醇的四个环骨架结构。
  再经过约20步反应转化为「胆固醇」。
  
胆固醇合成的调控
---------------
胆固醇的合成：是一个复杂的「耗能」过程，严格调节胆固醇合成以补充食物中摄取的胆固醇对生物体而言是有利的。
●在哺乳动物中，胆固醇「合成」受细胞内胆固醇含量、ATP供应、以及激素（胰岛素和胰高血糖素）调控，限速步骤是【HMG-CoA还原酶】催化的反应。【HMG-CoA还原酶】是一个复杂的调控酶，酶活性调节分为「短期」（磷酸化/去磷酸化——修饰调控）和「长期」（酶基因的转录水平受胞内胆固醇水平的调控）两种调控方式。
●胆固醇在细胞内的合成「部位」是「细胞溶胶和光面内质网」，合成胆固醇的酶也主要存在于这两个部位。
●合成的胆固醇：
  可以作为血浆「脂蛋白」中的组成成分进入血浆；
  以「胆固醇脂」的形式贮存在小滴droplet中；
  用于「细胞膜」的结构组成；
  转化为「胆汁盐」；
  在肾上腺或性腺中转化为多种「类固醇激素」。
