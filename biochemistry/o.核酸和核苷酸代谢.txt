核酸的降解
~~~~~~~~~~
核酸代谢
========
●核酸的基本「结构单位」是核苷酸。遗传信息的复制、重组、转录和各种加工变化均需通过「核酸代谢」才能实现，而核酸代谢与核苷酸代谢紧密关联。
●核酸降解产生核苷酸，核苷酸还能进一步降解。核苷酸的分解产物「核苷」和「碱基」可以通过「补救途径」加以再利用。
●生物体内，核苷酸可由少数化合物「从头合成」：核苷酸生物合成受「反馈调节」。
●核苷酸代谢和aa代谢相互密切相关：它们的合成有共同的氨供体和一碳供体，也有类似的代谢产物；有些核苷酸参与氨基酸的合成，有些氨基酸参与核苷酸的合成，两者相伴而成。
●核苷酸代谢还与其它物质代谢相关联：如有些「辅酶」含有腺苷酸、由ATP供给。
●核苷酸是重要生命分子，其「代谢异常」将造成疾病。

●「核苷酸」主要有以下几个方面的作用：
  1.是「核酸」生物合成的前体；
  2.ATP是生物新陈代谢通用的「能量载体」；GTP是推动某些重要生命过程的「能量供体」；
  3.「核苷酸衍生物」是许多生物合成反应的活性中间物；如UDP-Glc、CDP-二脂酰甘油分别是Glc和磷酸甘油酯合成的中间产物。
  4.腺苷酸和腺苷是「重要辅酶的辅助因子」的组分：如辅酶A、辅酶B12、SAM、NAD、FAD等。
  5.核苷酸作为「信号分子」，在调节细胞功能和基因表达中起重要作用：如cAMP、cGMP是信号转导的第二信使：腺苷酰基、尿苷酰基是酶活性调节的「共价修饰基团」等。

核酸的分解
==========
●动物和异氧型微生物，可以分泌【消化酶类】来分解食物或体外的核蛋白和核酸类物质，已获得各种核苷酸。
●所有生物的细胞都含有与「核酸代谢」有关的酶类，能够分解细胞内的各种核酸，促使外界入侵核酸或细胞内无用核酸的分解。
●核苷酸「水解」脱去磷酸生成核苷，核苷再分解生成含氮碱（嘌呤碱和嘧啶碱）和戊糖。
●核苷酸的「水解产物」核苷和碱基可以通过「补救途径」加以再利用均可被细胞吸收和利用。
●在体内，核酸水解产物「戊糖」可参加戊糖代谢：含氮碱（嘌呤碱和嘧啶碱）还可以进一步分解。
●核酸是核苷酸以「3',5'-磷酸二酯键」连接而成的大分子化合物。
●核酸分解的第一步是「水解」连接核苷酸之间的磷酸二酯键，生成低级多核苷酸和单核苷酸；由【核酸酶】进行催化，包括【RNase】和【DNase】。核酸酶中能够水解核酸「分子内」的磷酸二酯键的为【内切核酸酶】（endonuclease）；从核酸「一端」逐步水解下核苷酸的酶为【外切核酸酶】（exonuclease）。
●细胞内DNA含量相当稳定，RNA含量变化显著。但是【DNase】含量在细胞内却非常高，生物学功能可能在于消除异常的、无用的或外来的DNA，以维持细胞遗传性的稳定；或用于细胞凋亡和自溶。
●「核酸酶」的作用：清除细胞不需要的内、外源核酸：参与DNA的合成与修复、RNA合成后剪接等重要基因复制和表达过程；消耗降解食物中的核酸以利于吸收；体外重组DNA技术中的重要工具酶等。

●核苷酸「水解」下磷酸成为「核苷」，生物体内广泛存在的【磷酸单脂酶】或【核苷酸酶】可以催化这个反应。
●非特异性的【磷酸单脂酶】对一切核苷酸都能起作用，无论磷酸基在核苷的2',3',5'位置上都能被水解下来。某些特异性强的【磷酸单脂酶】只能水解3'-核苷酸或5'-核苷酸，则分别称为【3'-核苷酸酶】或【5'-核苷酸酶】。

●核苷经「核苷酶」（nucleosidase）作用分解为碱基（嘌呤碱和嘧啶碱）和戊糖。分解核苷的酶有两类：
  一类是【核苷磷酸化酶】，形成碱基和戊糖磷酸酯；
  另一类是【核苷水解酶】，形成碱基和戊糖；
●【核苷磷酸化酶】：存在比较「广泛」，所催化的反应是「可逆」的。
●【核苷水解酶】：主要存在于植物和微生物体内，并且只能对「核糖核苷」作用，对「脱氧核糖核苷」没有作用，反应是「不可逆」的。
●核苷酸的降解产物嘌呤碱和嘧啶碱再继续进行分解。

●不同种类的生物分解「嘌呤碱」的能力不一样，代谢产物也各不相同。人、猿类和排尿酸的动物以「尿酸」作为嘌呤碱代谢的最终产物，直至最后分解为CO₂和氨。
●嘌呤碱分解：首先是在各种【脱氨酶】的作用下水解脱去氨基。「腺嘌呤和鸟嘌呤」水解脱氨分别生成「次黄嘌呤和黄嘌呤」。脱氨反应也可以在「核苷或核苷酸」的水平上进行。
●动物组织中，【腺嘌呤脱氨酶】含量极少，而【腺嘌呤核苷酸脱羧酶】的活性较高。【鸟嘌呤脱氨酶】分布较广。

●次黄嘌呤和黄嘌呤在【黄嘌呤氧化酶】的作用下氧化形成「尿酸」。
●尿酸的进一步分解代谢随物种的不同而异。尿酸在【尿酸氧化酶】作用下生成「尿囊酸」，是除人及猿类以外其它哺乳动物嘌呤代谢的排泄物。
●对许多鱼类和两栖类，「尿囊酸」在【尿囊酸酶】作用下水解生成「尿素」和「乙醛酸」。
●某些低等动物还能将尿素分解为CO₂和氨。

●【黄嘌呤氧化酶】是一种「复合黄素酶」，由2个相同亚基组成。每个亚基为145kD，含有一个FAD、一个钼辅因子和两个不同的铁硫中心。
●黄嘌呤氧化是一个很复杂的过程，要求「分子氧」为电子受体，还原产物为「过氧化氢」，随即被【过氧化物酶】分解为O₂和H₂O；进入尿酸的O₂来自H₂O。

●「尿酸」的水溶性较差，尿酸过多容易引起结晶而沉积于关节、软组织、软骨及肾等处（结石），导致痛风（高尿酸血症）。
●结构和「次黄嘌呤」很相似的「别嘌呤醇」对【黄嘌呤氧化酶】有很强的「抑制」作用，用来治疗由于尿酸在体内过量积累导致的「痛风」。经「别嘌呤醇」治疗的患者排泄次黄嘌呤、黄嘌呤代替尿酸。

●不同物种对「嘌呤碱」的分解过程也不完全一样。
●具有氨基的嘧啶需要先「水解」脱去「氨基」，如「胞嘧啶」脱氨生成「尿嘧啶」。在人和某些动物体内，脱氨过程也可能是在「核苷或核苷酸」的水平上进行的。
●尿嘧啶经「还原」生成「二氢尿嘧啶」，并水解开环形成β-脲基丙酸，再水解成β-丙氨酸、CO₂和NH₃。

●「胸腺嘧啶」T的分解代谢和尿嘧啶相似，经还原生成「二氢胸腺嘧啶」，并水解开环形成「β-脲基异丁酸」，再水解形成β-氨基异丁酸、CO₂和NH₃。

●嘧啶碱分解代谢后产生的「β-丙氨酸和β-氨基异丁酸」可以进一步通过「转氨」作用脱去氨基后，参加有机酸的代谢：与「尿素循环」、「EMP」、「三羧酸循环」相联系。

核苷酸生物合成
~~~~~~~~~~~~~
●核苷酸生物合成由两条途径：「从头合成」途径和「补救」途径。
●Ⅰ.「补救」途径：生物体内对外源或内源核苷酸降解产生的碱基和核苷可以重新利用，通过【磷酸核糖转移酶】或【核苷激酶】作用，由碱基或核苷形成核苷酸。
●Ⅱ.「从头合成」途径：无论动物、植物或微生物，都能由一些简单的「前体」物质合成核苷酸。

嘌呤核糖核苷酸生物合成
====================
●用同位素标记的化合物进行实验，证明生物体内能利用CO₂、甲酸盐、谷氨酰胺、天冬氨酸、和甘氨酸作为前体分子来合成「嘌呤环」。
●「嘌呤环」中的N1来自Asp的氨基；C2和C8来自「N¹⁰-甲酰四氢叶酸」(formate)，甲酰基由「甲酸盐」提供；N3&N9来自Gln的酰胺基；C4&C5&N7来自Gly；C6来自CO₂。

●在生物体内，并不是先合成「嘌呤碱」，再与核糖和磷酸结合成核苷酸；而是从「5-磷酸核糖焦磷酸」开始，经过一些列酶促反应，生成「IMP」，再转变为其它嘌呤核苷酸（AMP或GMP）。
●肝是体内「从头合成」嘌呤核苷酸的主要器官，其次是小肠和胸腺，而脑和骨髓则无法进行此合成途径。

次嘌呤核苷酸（IMP）生物合成
---------------------
●「IMP」的合成途径是一系列连续的酶促反应过程，共有「10」步反应。
  ◆首先是「5-磷酸核糖焦磷酸」（PRPP）供给核苷酸的「磷酸核糖」部分，在PRPP基础上再完成「嘌呤环」的装配。在体内，PRPP由5-磷酸核糖与ATP作用产生，由【磷酸核糖焦磷酸激酶】催化反应，将焦磷酸基作为一个单位直接转移给5-磷酸核糖分子C1羟基上。
  ◆「PRPP」与「Gln」反应，经【酰胺转移酶】催化，生成「5-磷酸核糖胺」，并使原来的α-构型转变为β-构型。「5-磷酸核糖胺」和Gly在有ATP供能的情况下，合成「甘氨酰胺核糖核苷酸」，经「甲酰化」生成「甲酰甘氨酸酰胺核糖核苷酸」。甲酰基的供体是「N¹⁰-甲酰四氢叶酸」。
  ◆「甲酰甘氨酸酰胺核糖核苷酸」再由Gln提供「酰胺基」，并有ATP提供能量的情况下，转变为「甲酰甘氨脒」核糖核苷酸。这一步反应可被与Gln结构类似的「重氮Ser」等不可逆抑制，由抗癌作用，但副作用大。
  ◆在ATP存在时，「甲酰甘氨脒」核糖核苷酸经「合成酶」催化成环，形成「5-氨基咪唑」核苷酸，完成嘌呤碱的第一个环的合成。
  ◆然后在【羧化酶】作用下，加入羧基；然后在ATP帮助下，与「Asp」缩合形成「5-氨基咪唑-4-N-琥珀基-氨甲酰」核苷酸。
  ◆「5-氨基咪唑-4-N-琥珀基-氨甲酰」核苷酸由【裂解酶】催化「裂解」，脱去延胡索酸；在以「N¹⁰-甲酰四氢叶酸」提供甲酰基的情况下，甲酰化生成「5-甲酰胺基咪唑-4-羧酰胺」核苷酸。
  ◆在【IMP合成酶】催化下，脱水环化，形成嘌呤碱第二个环，生成「IMP」。
●IMP的全部酶促反应过程是「不可逆」的。
●合成过程总共消耗了「7个」高能磷酸基推动反应的进行，由「ATP」提供。
●IMP合成途径的酶在细胞中组成一个大的「多酶复合物」，使得一个酶促反应的产物直接进入下一个反应，防止中间产物的扩散丢失，也防止不稳定中间产物的分解。

●生物体内由「IMP」氨基化生成「AMP」，分为2步进行：IMP在GTP提供能量的条件下和Asp合成「腺苷酸琥珀酸」，随即在【裂解酶】催化下裂解，形成「AMP」和延胡索酸。
●GMP的合成：IMP经【IMP脱氢酶】催化氧化生成黄嘌呤核苷酸（XMP），再经「氨基化」生成GMP。
●细菌直接以「氨」作为氨基供体，而动物以Gln的酰氨基为氨基供体。氨基化需要ATP为能量供体。
●AMP的合成需要GTP，而GMP的合成需要ATP提供能量；通过这种方式起「协调」两种核苷酸水平的作用。

●生物体除能以「简单前体」物质“从头合成”核苷酸外，还能由已有的「碱基和核苷」合成核苷酸，是对核苷酸代谢的一种“回收”途径或“补救”途径，以便「更经济」的利用已有的成分。
●嘌呤碱与「5-磷酸核糖焦磷酸」在【磷酸核糖转移酶】的催化下，形成「嘌呤」核苷酸。
●目前已经分离出两种具有不同特异性的酶：【腺嘌呤磷酸核糖转移酶】（APRT）和【次黄嘌呤-鸟嘌呤磷酸核糖转移酶】（HGPRT）。

●重要补救途径：
  腺嘌呤 + PRPP --[APRT]-->  AMP + PPi
  次黄嘌呤 + PRPP --[HGPRT]--> IMP + PPi
  鸟嘌呤 + PRPP --[HGPRT]--> GMP + PPi
  
●在生物体中，还发现有【腺苷激酶】的存在，但缺乏其他嘌呤核苷的激酶。说明在AMP合成中，可能还存在【核苷激酶】途径进行AMP的回收。
☆腺嘌呤 + 1-磷酸核糖 -[核苷磷酸化酶]-> Pi + 腺嘌呤核苷
☆腺嘌呤核苷 ATP -[腺苷激酶]-> ADP + AMP
●嘌呤核苷酸的「从头合成」和「回收」途径之间通常存在「协调和平衡」。「5-磷酸核糖焦磷酸」是回收途径的主要底物，回收途径的不顺畅会导致「5-磷酸核糖焦磷酸」的大量积累，引起「嘌呤」核苷酸从头合成和嘌呤降解的增加；并积累尿酸，造成肾结石和痛风。
●Lesch-Nyhan综合症：一种与X染色体连锁的遗传代谢病，多见于男性。患者先天缺乏【HGPRT】，使得鸟嘌呤和次黄嘌呤回收途径受阻，导致PRPP积累，促使核苷酸从头合成以及降解的增加，导致尿酸过量产生，并会导致神经系统损伤。

嘌呤核苷酸合成的调节
--------------------
●嘌呤核苷酸的「从头合成」受其两个终产物「酰苷酸」和「鸟苷酸」的反馈控制。
  1.PRPP是合成反应的最初底物，它的合成受底物ADP的反馈抑制。
  2.在合成途径的第一步，「氨基」转移到PRPP上形成「5-磷酸核糖胺」，催化的酶是一种「变构酶」，受终产物IMP、AMP和GMP所抑制。
  3.IMP形成后的分子途径的第一步：AMP的积累仅抑制AMP的形成，而GMP的积累抑制GMP的合成。
  
嘧啶核苷酸合成
-------------
●与嘌呤核苷酸合成中不先合成「嘌呤碱」不同；从头合成「嘧啶核苷酸」时，首先合成的是「嘧啶环」，再与「磷酸核糖」结合形成「乳清苷酸」，然后生成「UMP」。其它嘌呤核苷酸则由UMP转变而来。
●嘌呤核苷酸的「嘧啶环」是由「氨甲酰磷酸」和Asp合成的。

●与嘌呤核苷酸合成中不先合成「嘌呤碱」不同；从头合成「嘧啶核苷酸」时，线粒体参与「尿素」合成；酶Ⅱ存在于细胞质中，参与「嘧啶环」的合成。
●用于形成「嘧啶环」的「氨甲酰磷酸」需要由Gln作为「氨基」的供体，与HCO₃⁻和ATP反应；每合成1分子的「氨甲酰磷酸」需要消耗「2分子的ATP」。
●反应机制：
  1.首先HCO₃⁻被ATP活化，生成「羰基磷酸」；
  2.Gln「转氨基」给羰基磷酸，形成「氨基甲酸脂」；
  3.再由ATP提供磷酸基磷酸化，形成「氨甲酰磷酸」；

尿嘧啶核苷酸合成
---------------
●「氨甲酰磷酸」形成后，在【Asp转氨甲酰酶】的作用下，首先与「Asp」，合成「氨甲酰Asp」；
●「氨甲酰Asp」通过可逆的「环化」脱水作用成环，转变为「二氢乳清酸」；催化的酶是【二氢乳清酸酶】；
●二氢乳清酸在【脱氢酶】的作用下，脱氢氧化生成「乳清酸」。「乳清酸」是合成UMP的重要中间产物，此时已形成「嘧啶环」。
●乳清酸再与PRPP作用生成「乳清苷酸」，「脱羧」后形成UMP。

●与IMP的生物合成相似，在动物中UMP合成途径的前三个酶（【氨甲酰磷酸合成酶Ⅱ】、【Asp转甲氨酰酶】和【二氢乳清酸酶】）组成一个【多功能酶】，缩写为「CAD」，它由3个相同的多肽链亚基组成，每个亚基都包含全部三个反应中心的活性中心；最后2步反应的酶（【乳清酸磷酸核糖转移酶】、【乳清核苷酸脱羧酶】）也融合成一条多肽链，简称【UMP合成酶】。
●【UMP合成酶】有缺陷时，即患「乳清酸尿症」。患者需要在饮食中提供尿苷或胞苷，或抑制氨甲酰磷酸合成酶Ⅱ，以减少乳清酸的合成。

胞嘧啶核苷酸合成
---------------
●由UMP合成CMP，需要在UTP的水平上进行。尿嘧啶、尿嘧啶核苷和UMP都不能通过「氨基化」形成相应的胞嘧啶化合物。
●UTP由UMP在相应【激酶】作用下，经2步ATP转移「磷酸基」生成，先从UMP形成UDP，后形成UTP。
●催化UMP转变为UDP的酶是特异的【UMP激酶】，而催化UDP转化为UTP的酶是特异性较广的【核苷二磷酸激酶】
☆UMP + ATP -[UMP激酶&Mg²⁺]-> UDP + ADP
☆UDP + ATP -[核苷二磷酸激酶&Mg²⁺]-> UTP + ADP
●UTP「氨基化」后生成CMP；在细菌中，「UTP」可以直接和「氨」作用；在动物组织中，则需要Gln供给氨基。反应需要由ATP供给能量，由【CTP合酶】催化进行。
☆UTP + ATP + H₂O -[CTP合酶]-> CTP + ADP + Pi

嘧啶核苷酸补救合成途径
---------------------
生物体对内源和外源产生的嘧啶碱和核苷的重新利用
●尿嘧啶转变为UMP可以通过：
  1.「尿嘧啶」直接与PRPP反应；
  2.或者先与「1-磷酸核糖」反应生成「尿嘧啶核苷」，再在【尿苷激酶】作用下「磷酸化」形成UMP；
●胞嘧啶只能通过【激酶】途径生成CMP。【尿苷激酶】也能催化「胞苷」被ATP磷酸化为CMP。
1.☆尿嘧啶 + PRPP -[嘧啶磷酸核糖转移酶]-> UMP + PPi
2.1.☆尿嘧啶 + 1-磷酸核苷 -[尿苷磷酸化酶]-> 尿嘧啶核苷 + Pi
2.2.☆尿嘧啶核苷 + PRPP -[尿苷激酶]-> UMP + ADP

嘧啶核苷酸合成的调节
--------------------
●嘧啶核苷酸合成受「产物」的反馈调节；但生物不同，具体的机制也各不相同。
●〔细菌〕嘧啶核苷酸合成主要通过对【Asp转氨甲酰酶，ATCase】活性调控来进行的。该酶是一个调节酶，由6个催化亚基和6个调节亚基组成。大肠杆菌对ATCase调节的效应物是「终产物」CTP。当全部调节亚基都没有与CTP结合时，酶的活性最高；CTP积累并结合在调节亚基上，通过变构效应使催化亚基由活性构象转变为无活性构象。但在另一些细菌中，变构效应物是UTP。
●在〔动物〕中，ATCase不是调节酶，嘧啶核苷酸合成的控制点是更早的【氨甲酰磷酸合成酶Ⅱ】，它受到UMP的反馈抑制。

脱氧核苷酸的生物合成
~~~~~~~~~~~~~~~~~~~
●生物体内的「脱氧」核糖核苷酸可以由核糖核苷酸「还原」形成。腺嘌呤、鸟嘌呤和胞嘧啶核苷酸经还原，将其中「核糖C2位」上的氧脱去，即形成相应的「脱氧」核糖核苷酸。「胸腺嘧啶」脱氧核糖核苷酸的形成则需要经过两个步骤，首先由UMP还原形成「尿嘧啶脱氧核糖」核苷酸，然后再经过尿嘧啶「甲基化」转变为胸腺嘧啶。
●核糖核苷酸的还原由【核糖核苷酸还原酶】催化进行，是在「核苷二磷酸」水平上进行的。核糖核苷酸必须转化为「二磷酸」核苷酸（NDP），然后再还原为「脱氧」核苷二磷酸（dNDP）。

●【核糖核苷酸还原酶】由催化亚基R1和自由基产生亚基R2所组成，它们分开没有酶活性，只有合在一起并由Mg²⁺存在时才具有催化活性。
●R1亚基含有两条相同的多肽链（α），每个α上有两个「变构调节位点」和一对参与还原反应的巯基；ATP、dATP、dGTP、dTTP是还原酶的变构效应物。
●R2也有两条相同的多肽链（β），每个β上有一个「酪氨酰自由基」和一个「铁」中心。「铁」中心含2个Fe³⁺，由氧离子桥接在一起，「产生和稳定」酪氨酸自由基。
●R1和R2亚基的交界处形成催化反应的活性位点。

●目前已发现的【核糖核苷酸还原酶】可分为4种类型，他们的区别主要是在于提供活性位点「自由基」的基团和用于产生它的「辅助」因子的不同。
●类型Ⅰ是含有「酪氨酸自由基」和「双氧铁」中心的酶，如大肠杆菌的酶，在酪氨酰自由基被淬灭后要求「氧」进行再生，因此必需在「有氧条件」下才具有功能。
●类型Ⅱ含有「5'-脱氧腺苷钴胺素」，而不是双核铁中心，存在于某些微生物中。
●类型Ⅲ含有「铁硫」簇，并且活化要求NADPH和S-腺苷硫胺酸。以「腺苷三磷酸」作为底物，而不是通常的核苷二磷酸，可适用于「厌氧」环境下反应。
●类型Ⅳ含有「双核Mn」中心，存在于某些微生物中。
●生物在进化过程中产生不同种类的【核糖核苷酸还原酶】，使得不同环境下均能合成DNA前体，它们基本过程和机制完全相同，反映了该反应在核苷酸代谢中的重要性。

●【核糖核苷酸还原酶】的作用机制：
⑴还原酶R2亚基上的Cys439的H原子被Tyr122「酪氨酰」自由基的孤电子置换，使得Cys439形成高活性的硫自由基；它吸引底物NDP上C3位H，产生「NDP-C3碳自由基」。
⑵⑶该自由基促使C2-OH从R1亚基上的一个Cys获得H并脱水，产生「NDP-C2阳离子」。
⑷「NDP-C2阳离子」从R1上另一个Cys上获得氢负离子，并使得R1亚基上的Cys255和Cys462形成「双硫键」。
⑸「NDP-C3碳自由基」再次从R2亚基上的Cys439的捕获H原子，生成「dNDP」离开酶。
⑹酶的二硫键由还原性「硫氧还原蛋白」还原。

●H的传递：
  「核糖」核苷酸被酶还原成「脱氧核糖」核苷酸需要提供2个H，酶失去2个H形成「双硫键」，需要「氢供体」使其恢复。
  氢原子的最终供体是「NADPH」，氢原子经「氢携带」蛋白转移给还原酶，最终传递到底物核苷酸上。
●「硫氧还」蛋白是一种广泛参与氧化还原反应的小分子蛋白质，含有一对巯基，给出2个H后成为氧化型；再在【硫氧还蛋白还原酶】作用下被NADPH还原，其还原酶是一种含有FAD的黄素酶。
●另一种「氢携带蛋白」是「谷氧还蛋白」。【谷氧还蛋白还原酶】结合2分子的「谷胱甘肽GSH」，可以还原谷氧还蛋白。【谷胱甘肽还原酶】也是一种黄素酶，从NADPH获得氢，从而还原「谷胱甘肽」。

【核糖核苷酸还原酶】的调节
===================
●【核糖核苷酸还原酶】受两方面的精确调控：
  1.核糖核苷酸和脱氧核糖核苷酸「供求」关系的调节；
  2.「4种脱氧核苷二磷酸」之间维持平衡；
●【还原酶】R1亚基有两个「酶活性调节部位」和「两个底物特异性调节位点」。
●「酶活性调节部位」可根据细胞内「核糖核苷酸和脱氧核糖核苷酸水平」打开或关闭酶的活性：
  dATP的浓度作为可以反映四种dNTP水平的指标；
  ATP的浓度则反映了核糖核苷酸和供能的水平。
  当ATP结合于酶活性调节位点时，酶被激活，为「正」效应物；dATP结合则酶被抑制，为「负」效应物。它们竞争酶的同一位点。效应物和酶结合，改变酶的结构，影响酶的活性。
  dATP过量时，与【还原酶】结合，促使有活性的α2β2聚合成无活性的「环状α4β4」.

●「底物特异性调节位点」，可被ATP、dATP、dGTP和dTTP结合，哪种核苷酸结合到该位点决定了「酶底物特异性」。
  如「ATP或ADP」结合在调节位点上，【还原酶】优先结合「嘧啶」核苷酸底物（UDP或CDP），并使其还原为dUDP和dCDP；
  如「dTTP」结合在调节位点上，【还原酶】优先结合GDP，并抑制UDP和CDP还原；
  如「dGTP」结合在调节位点上，【还原酶】优先结合ADP，并抑制UDP和CDP以及GDP还原；
●通过这种模式，控制4种脱氧核糖核苷酸得以达到「平衡」。

胸腺嘧啶核苷酸（dTMP）合成
====================
●dTMP是脱氧核糖苷酸的组成部分，由「dUMP」经「甲基化」而生成的。催化的酶为【胸腺嘧啶核苷酸合成酶】。甲基的供体是「N⁵ ,N¹⁰-亚甲基四氢叶酸」。
●「N⁵ ,N¹⁰-亚甲基四氢叶酸」提供甲基后，自身变成「二氢叶酸」，经【二氢叶酸还原酶】催化，形成「四氢叶酸」，再与Ser反应恢复成「N⁵ ,N¹⁰-亚甲基四氢叶酸」。
☆dUMP + 「N⁵ ,N¹⁰-亚甲基四氢叶酸」 --[胸腺嘧啶核苷酸合酶]--> 二氢叶酸 + dTMP
☆7,8-二氢叶酸 + NADPH + H⁺  <--[二氢叶酸还原酶]--> 5,6,7,8-四氢叶酸 + NADP⁺
☆Ser + 四氢叶酸 <--[Ser羟甲基转移酶]--> Gly + 「N⁵ ,N¹⁰-亚甲基四氢叶酸」 + H₂O

●至于合成「胸腺嘧啶」核苷酸所需要的底物dUMP，可以由「尿嘧啶」核苷二磷酸（UDP）「还原」成尿嘧啶脱氧核苷二磷酸（dUDP），经「磷酸化」形成尿嘧啶脱氧核苷三磷酸（dUTP），再经【尿嘧啶脱氧核苷三磷酸（dUTPase）】转变成尿嘧啶脱氧核苷「一磷酸」（dUMP）。
●或者由dCTP「脱氨」，形成dUTP，再转变为dUMP。
●何者为主在不同的生物体内可能不同。
●为防止「鸟苷酸」渗入DNA，细胞内dUTP一经合成即被【dUTPase】转变为dUMP，保持dUTP在一个很低的水平。
☆CDP->dCDP->dCTP --> dUTP --> dUMP --> dTMP
☆UDP->dUDP->dUTP --> dUTP --> dUMP --> dTMP

「脱氧核糖核苷酸」的合成也能通过「补救」途径，利用已有的碱基和核苷进行合成。
●「脱氧核糖核苷」由碱基和脱氧核糖-1-磷酸，在【嘌呤或嘧啶核苷酸磷酸化酶】的催化下形成。
●「脱氧核糖核苷」分别在特异的【脱氧核糖核苷激酶】和ATP的作用下，被磷酸化形成相应的「脱氧核糖核苷酸」
●体内不存在相应于【磷酸核糖转移酶】的【磷酸脱氧核糖转移酶】途径。
☆胸腺嘧啶 + 脱氧核糖-1-磷酸 --[胸腺嘧啶核苷酸磷酸化酶]--> 胸腺嘧啶核苷
☆胸腺嘧啶核苷 + ATP --[dT kinase]--> dTMP + ADP


辅酶核苷酸的合成
~~~~~~~~~~~~
●生物体内有很多「核苷酸衍生物」作为「辅酶」而发挥作用。
●其中最重要的有烟酰胺腺嘌呤二核苷酸（NAD），烟酰胺腺嘌呤二核苷酸磷酸（NADP）、黄素单核苷酸（FMN）、黄素腺嘌呤二核苷酸（FAD）以及辅酶A。这几种辅酶核苷酸都可以在体内自由存在。

●NAD和NADP是含有「烟酰胺」的两种腺嘌呤核苷酸的衍生物。它们是【脱氢酶】的辅酶，在生物氧化还原体系中起「氢传递体」的作用。
●NAD由一分子的「烟酰胺核苷酸」（NMN）和一分子的「腺嘌呤核苷酸」连接而成，NADP则是在NAD的腺嘌呤核糖的C2-OH上多了一个磷酸基。

●NAD和NADP的生物合成：
  1.烟酸先与「5-磷酸核糖焦磷酸」反应，产生「烟酸单核苷酸」，催化的酶为【烟酸单核苷酸焦磷酸化酶】；
  2.烟酸单核苷酸与ATP在【脱酰基-NAD焦磷酸酶】的催化下进行缩合反应，形成「脱酰胺-NAD」；
  3.「脱酰胺-NAD」酰胺化，利用「谷氨酰胺」为酰胺氮的供体，形成「NAD」，催化的酶为【NAD合成酶】；
  4.NAD由【NAD激酶】催化磷酸化，形成NADP；
  ☆烟酸 + 5-磷酸核糖焦磷酸 --[烟酸单核苷酸焦磷酸化酶]--> 烟酸单核苷酸 + PPi
  ☆烟酸单核苷酸 + ATP --[脱酰胺-NAD焦磷酸化酶]--> 脱酰胺-NAD + PPi
  ☆脱酰胺-NAD + ATP + Gln --[NAD合成酶]--> NAD + Glu + AMP + PPi
  ☆NAD + ATP --[NAD激酶]-->   NADP + ADP
  
●「黄素核苷酸」是核黄素的衍生物，又称为「异咯嗪(qin)」核苷酸，共有两种：
  1.黄素单核苷酸（FMN）；
  2.黄素腺嘌呤二核苷酸（FAD）；
●它们是许多氧化还原酶的辅基，以其「异咯嗪（黄素）」部分的氧化还原参与传递氢和电子的作用。
●FMN由「6,7-二甲基异咯嗪核醇」和磷酸所组成；
●FAD由一分子的FMN和一分子的「腺苷酸」联接而成。

●动植物和微生物都能利用「核黄素」以合成「黄素核苷酸」。核黄素在【黄素激酶】的催化下与ATP反应生成5-磷酸核黄素FMN，即异咯嗪单核苷酸。
☆核黄素 + ATP --[【黄素激酶】&Mg²⁺]--> FMN + ADP
●FMN又在【FAD焦磷酸酶】的作用下，与ATP反应生成FAD。反应释放的PPi完全来自ATP。
☆FMN + ATP --【FAD焦磷酸酶】--> FAD + PPi

●「辅酶A」分子中含有腺苷酸、泛素、巯基乙酸和磷酸，它们的连接方式为：3-磷酸-ADP-泛素-巯基乙胺。
●「辅酶A」是【酰基转移酶】的辅酶。
●「辅酶A」分子的生物合成是从「泛素」开始的。
  Step1：首先在【泛素激酶】的作用下，泛酸与ATP反应形成「4'-磷酸泛酸」。该激酶已经从动物、细菌和酵母中分别提取得到。
  ☆泛酸 + ATP --【泛素激酶】--> 4'-磷酸泛酸 + ADP
  Step2：「4'-磷酸泛酸」与「半胱氨酸」缩合产生「4'-磷酸泛酰半胱氨酸」，催化的酶为【磷酸泛酰半胱氨酸合成酶】。从细菌中提取到的合成酶必须以CTP提供能量，但从动物系统中提取的合成酶可用其他核苷三磷酸代替CTP。
  ☆4'-磷酸泛酸 + Cys + ATP  --【磷酸泛酰半胱氨酸合成酶】--> 4'-磷酸泛酰半胱氨酸 + ADP + Pi
  Step3：「4'-磷酸泛酰半胱氨酸」在【磷酸泛酸半胱氨酸脱羧酶】的作用下，脱去羧基，转变成「4'-磷酸泛酸巯基乙胺」。
  ☆4'-磷酸泛酰半胱氨酸 --【磷酸泛酸半胱氨酸脱羧酶】--> 4'-磷酸泛酸巯基乙胺 + CO₂
  Step4：「4'-磷酸泛酸巯基乙胺」 与 ATP缩合形成「脱磷酸辅酶A」，并释放无机焦磷酸。这是辅酶A生物合成过程中唯一的「可逆」反应，催化该反应的酶为【脱磷酸辅酶A焦磷酸化酶】。
  ☆4'-磷酸泛酸巯基乙胺 + ATP --【磷酸泛酸半胱氨酸脱羧酶】--> 脱磷酸辅酶A + AMP + PPi
  Step5：脱磷酸辅酶A在【脱磷酸辅酶A激酶】的催化下，被「磷酸化」形成「辅酶A」。磷酸基的供体必须是「ATP」，并且在有Cys时，活性最大。
  ☆脱磷酸辅酶A + ATP --【脱磷酸辅酶A激酶】--> 辅酶A + ADP
